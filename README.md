# MovieSite
单页在线影院网站设计与实现

本设计服务器端基于ThinkPHP 3.2.3
PHP 版本， 建议 PHP 5.3 以上环境

### 功能
 - [x] 用户注册/登录
 - [x] 电影检索
 - [x] 付费（虚拟）观看
 - [x] 观影记录
 - [x] 在线观看（使用[DPlayer](https://github.com/DIYgod/DPlayer)）
 - [ ] 后台管理
